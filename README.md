# Maestro — site + releases

Public home of [Maestro](https://maestroterminal.com/), a native macOS
terminal multiplexer for orchestrating many AI coding agents in parallel.

- **Website:** https://maestroterminal.com/ (served from this repo's root via GitHub Pages)
- **Downloads:** the [Releases](https://github.com/coldstar96/maestro-releases/releases) of this repo.
  Recommended install (app + CLI, strips quarantine):
  `curl -fsSL https://maestroterminal.com/install.sh | sh`
  Latest dmg (evergreen URL): https://github.com/coldstar96/maestro-releases/releases/latest/download/Maestro-arm64.dmg
- macOS · Apple Silicon. Unsigned build — first launch: right-click → Open.

The application source lives in a separate private repository; this repo hosts the site and
the binary releases only.
